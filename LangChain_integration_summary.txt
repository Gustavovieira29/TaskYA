Resumo das alterações para integração do LangChain

1. Atualização em app.py
- Incluído import de `load_dotenv` para carregar variáveis de ambiente.
- Adicionados imports LangChain:
  - `LLMChain`
  - `ChatOpenAI`
  - `ChatPromptTemplate`
  - `HumanMessagePromptTemplate`
- Inserido `load_dotenv()` no início do arquivo para ler a API key do ambiente.
- Criada a função `generate_task_summary(tasks)` que:
  - monta uma lista de tarefas em formato de texto;
  - configura um prompt do LangChain para gerar um resumo das tarefas;
  - utiliza `ChatOpenAI` com temperatura 0.3;
  - trata exceções e registra erros com `log_error`.
- Adicionado novo endpoint `/generate_summary` que:
  - lê as tarefas do banco;
  - ordena as tarefas;
  - chama `generate_task_summary` e retorna o texto como resposta plain text.

2. Atualização em requirements.txt
- Adicionada a dependência `langchain>=0.0.375`.

3. Atualização em templates/index.html
- Incluído um painel de resumo com LangChain.
- Adicionado botão `Gerar resumo com LangChain`.
- Adicionado `JavaScript` para chamar `/generate_summary` via `fetch` e exibir o resultado na página.

Observações
- A integração depende de chaves de API OpenAI corretamente configuradas no ambiente.
- O endpoint e o interface foram adicionados para demonstrar o uso básico do LangChain na aplicação.
